## Packages
recharts | Dashboard analytics and graphs
date-fns | Date manipulation and formatting
framer-motion | Smooth animations and page transitions
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging tailwind classes

## Notes
The application uses Replit Auth.
Razorpay integration is simulated on the frontend for the prototype but wired to backend endpoints.
Graphs require recharts.
Kanban board implemented with local state and mutation hooks.

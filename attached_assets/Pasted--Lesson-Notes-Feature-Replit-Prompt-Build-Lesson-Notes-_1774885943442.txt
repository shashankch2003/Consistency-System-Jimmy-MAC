# Lesson Notes Feature — Replit Prompt

Build **Lesson Notes** — A polished, practical note-taking feature where users create pages and write rich content using a TipTap-based editor with slash commands, headings (H1/H2/H3), bullet lists, numbered lists, toggle blocks, image embedding, undo/redo, and export/sharing. The editor must feel smooth and simple like Notion but only implement the most-used features listed below — no database-backed blocks, no drag-and-drop reordering, no nested pages.

### 2.1 Note CRUD Routes

**GET `/api/lesson-notes`** — Returns all non-archived notes for the user, ordered by `lastEditedAt` desc. Select only these columns: `id`, `title`, `emoji`, `coverColor`, `isFavorite`, `isArchived`, `sortOrder`, `lastEditedAt`, `createdAt`. Do NOT select `content` in this list query (it can be large).

```ts
app.get("/api/lesson-notes", isAuthenticated, async (req, res) => {
  const notes = await db.select({
    id: lessonNotes.id,
    title: lessonNotes.title,
    emoji: lessonNotes.emoji,
    coverColor: lessonNotes.coverColor,
    isFavorite: lessonNotes.isFavorite,
    isArchived: lessonNotes.isArchived,
    sortOrder: lessonNotes.sortOrder,
    lastEditedAt: lessonNotes.lastEditedAt,
    createdAt: lessonNotes.createdAt,
  }).from(lessonNotes)
    .where(and(eq(lessonNotes.userId, req.user.claims.sub), eq(lessonNotes.isArchived, false)))
    .orderBy(desc(lessonNotes.lastEditedAt));
  res.json(notes);
});
```

**GET `/api/lesson-notes/:id`** — Returns a single note with full content. Validates the note belongs to the user. Returns 404 with `{ error: "Note not found" }` if not found or not owned by user.

```ts
app.get("/api/lesson-notes/:id", isAuthenticated, async (req, res) => {
  const noteId = parseInt(req.params.id);
  if (isNaN(noteId)) return res.status(400).json({ error: "Invalid note ID" });
  const [note] = await db.select().from(lessonNotes)
    .where(and(eq(lessonNotes.id, noteId), eq(lessonNotes.userId, req.user.claims.sub)));
  if (!note) return res.status(404).json({ error: "Note not found" });
  res.json(note);
});
```

**POST `/api/lesson-notes`** — Creates a new note. Sets `userId` from session. Validates body:

```ts
z.object({
  title: z.string().min(1).max(200).optional(),
  emoji: z.string().max(10).optional(),
  coverColor: z.string().max(20).optional(),
})
```

Returns the full created note object. Sets `sortOrder` to the current count of the user's notes (so newest goes to end).

```ts
app.post("/api/lesson-notes", isAuthenticated, async (req, res) => {
  const schema = z.object({
    title: z.string().min(1).max(200).optional(),
    emoji: z.string().max(10).optional(),
    coverColor: z.string().max(20).optional(),
  });
  const parsed = schema.parse(req.body);
  const userId = req.user.claims.sub;
  const [countResult] = await db.select({ count: sql<number>`count(*)` }).from(lessonNotes).where(eq(lessonNotes.userId, userId));
  const sortOrder = Number(countResult.count);
  const [note] = await db.insert(lessonNotes).values({
    userId,
    title: parsed.title || "Untitled",
    emoji: parsed.emoji || "📝",
    coverColor: parsed.coverColor || "#6366f1",
    sortOrder,
    content: "",
  }).returning();
  res.json(note);
});
```

**PATCH `/api/lesson-notes/:id`** — Updates a note. Validates owner. Before updating `content`, save the current version to `lessonNoteVersions` (only if content actually changed). Update `lastEditedAt` to `new Date()` on every PATCH.

Validates body:
```ts
z.object({
  title: z.string().min(1).max(200).optional(),
  content: z.string().optional(),
  emoji: z.string().max(10).optional(),
  coverColor: z.string().max(20).optional(),
  isFavorite: z.boolean().optional(),
  isPublic: z.boolean().optional(),
  sortOrder: z.number().int().min(0).optional(),
})
```

Implementation:
```ts
app.patch("/api/lesson-notes/:id", isAuthenticated, async (req, res) => {
  const noteId = parseInt(req.params.id);
  if (isNaN(noteId)) return res.status(400).json({ error: "Invalid note ID" });
  const userId = req.user.claims.sub;

  const [existingNote] = await db.select().from(lessonNotes)
    .where(and(eq(lessonNotes.id, noteId), eq(lessonNotes.userId, userId)));
  if (!existingNote) return res.status(404).json({ error: "Note not found" });

  const schema = z.object({
    title: z.string().min(1).max(200).optional(),
    content: z.string().optional(),
    emoji: z.string().max(10).optional(),
    coverColor: z.string().max(20).optional(),
    isFavorite: z.boolean().optional(),
    isPublic: z.boolean().optional(),
    sortOrder: z.number().int().min(0).optional(),
  });
  const parsed = schema.parse(req.body);

  if (parsed.content !== undefined && parsed.content !== existingNote.content) {
    await db.insert(lessonNoteVersions).values({
      userId,
      noteId,
      title: existingNote.title,
      content: existingNote.content,
    });
  }

  if (parsed.isPublic === true && !existingNote.publicSlug) {
    const slug = `${noteId}-${Date.now().toString(36)}`;
    (parsed as any).publicSlug = slug;
  }

  const [updated] = await db.update(lessonNotes)
    .set({ ...parsed, lastEditedAt: new Date() })
    .where(and(eq(lessonNotes.id, noteId), eq(lessonNotes.userId, userId)))
    .returning();
  res.json(updated);
});
```

**DELETE `/api/lesson-notes/:id`** — Soft-deletes by setting `isArchived` to `true`. Validates owner.

```ts
app.delete("/api/lesson-notes/:id", isAuthenticated, async (req, res) => {
  const noteId = parseInt(req.params.id);
  if (isNaN(noteId)) return res.status(400).json({ error: "Invalid note ID" });
  const [updated] = await db.update(lessonNotes)
    .set({ isArchived: true })
    .where(and(eq(lessonNotes.id, noteId), eq(lessonNotes.userId, req.user.claims.sub)))
    .returning();
  if (!updated) return res.status(404).json({ error: "Note not found" });
  res.json({ success: true });
});
```

### 2.2 Version History Routes

**GET `/api/lesson-notes/:id/versions`** — Returns all versions for a note, ordered by `createdAt` desc. Limit to 50 most recent. Validates the note belongs to the user.

```ts
app.get("/api/lesson-notes/:id/versions", isAuthenticated, async (req, res) => {
  const noteId = parseInt(req.params.id);
  if (isNaN(noteId)) return res.status(400).json({ error: "Invalid note ID" });
  const [note] = await db.select({ id: lessonNotes.id }).from(lessonNotes)
    .where(and(eq(lessonNotes.id, noteId), eq(lessonNotes.userId, req.user.claims.sub)));
  if (!note) return res.status(404).json({ error: "Note not found" });
  const versions = await db.select().from(lessonNoteVersions)
    .where(and(eq(lessonNoteVersions.noteId, noteId), eq(lessonNoteVersions.userId, req.user.claims.sub)))
    .orderBy(desc(lessonNoteVersions.createdAt))
    .limit(50);
  res.json(versions);
});
```

**POST `/api/lesson-notes/:id/versions/:versionId/restore`** — Restores a note to a previous version. Saves the current state as a new version first, then overwrites content and title from the selected version.

```ts
app.post("/api/lesson-notes/:id/versions/:versionId/restore", isAuthenticated, async (req, res) => {
  const noteId = parseInt(req.params.id);
  const versionId = parseInt(req.params.versionId);
  if (isNaN(noteId) || isNaN(versionId)) return res.status(400).json({ error: "Invalid ID" });
  const userId = req.user.claims.sub;

  const [note] = await db.select().from(lessonNotes)
    .where(and(eq(lessonNotes.id, noteId), eq(lessonNotes.userId, userId)));
  if (!note) return res.status(404).json({ error: "Note not found" });

  const [version] = await db.select().from(lessonNoteVersions)
    .where(and(eq(lessonNoteVersions.id, versionId), eq(lessonNoteVersions.noteId, noteId), eq(lessonNoteVersions.userId, userId)));
  if (!version) return res.status(404).json({ error: "Version not found" });

  await db.insert(lessonNoteVersions).values({
    userId,
    noteId,
    title: note.title,
    content: note.content,
  });

  const [updated] = await db.update(lessonNotes)
    .set({ title: version.title, content: version.content, lastEditedAt: new Date() })
    .where(eq(lessonNotes.id, noteId))
    .returning();
  res.json(updated);
});
```

### 2.3 Image Upload Route

**POST `/api/lesson-notes/upload-image`** — Accepts a multipart form upload with a single file field named `image`. Saves the file to the server and returns a URL. Use `multer` for parsing.

```ts
import multer from "multer";
import path from "path";
import fs from "fs";
import crypto from "crypto";

const uploadDir = path.join(process.cwd(), "uploads", "lesson-notes");
if (!fs.existsSync(uploadDir)) fs.mkdirSync(uploadDir, { recursive: true });

const lessonNotesUpload = multer({
  storage: multer.diskStorage({
    destination: uploadDir,
    filename: (req, file, cb) => {
      const ext = path.extname(file.originalname);
      const name = crypto.randomBytes(16).toString("hex");
      cb(null, `${name}${ext}`);
    },
  }),
  limits: { fileSize: 5 * 1024 * 1024 },
  fileFilter: (req, file, cb) => {
    const allowed = [".jpg", ".jpeg", ".png", ".gif", ".webp", ".svg"];
    const ext = path.extname(file.originalname).toLowerCase();
    if (allowed.includes(ext)) cb(null, true);
    else cb(new Error("Only image files are allowed"));
  },
});

app.post("/api/lesson-notes/upload-image", isAuthenticated, lessonNotesUpload.single("image"), (req, res) => {
  if (!req.file) return res.status(400).json({ error: "No image file provided" });
  const url = `/uploads/lesson-notes/${req.file.filename}`;
  res.json({ url });
});
```

### 2.5 Export Route

**GET `/api/lesson-notes/:id/export?format=html|markdown`** — Exports the note content. For HTML, returns the raw TipTap HTML content wrapped in a styled HTML document. For Markdown, converts the HTML to Markdown using `turndown` library on the server.

```ts
import TurndownService from "turndown";

app.get("/api/lesson-notes/:id/export", isAuthenticated, async (req, res) => {
  const noteId = parseInt(req.params.id);
  if (isNaN(noteId)) return res.status(400).json({ error: "Invalid note ID" });
  const format = req.query.format as string;
  if (!format || !["html", "markdown"].includes(format)) {
    return res.status(400).json({ error: "Format must be html or markdown" });
  }
  const [note] = await db.select().from(lessonNotes)
    .where(and(eq(lessonNotes.id, noteId), eq(lessonNotes.userId, req.user.claims.sub)));
  if (!note) return res.status(404).json({ error: "Note not found" });

  if (format === "html") {
    const html = `<!DOCTYPE html><html><head><meta charset="utf-8"><title>${note.title}</title><style>body{font-family:-apple-system,BlinkMacSystemFont,'Segoe UI',sans-serif;max-width:800px;margin:40px auto;padding:0 20px;line-height:1.6;color:#1a1a1a}h1{font-size:2em;margin-top:1em}h2{font-size:1.5em;margin-top:0.8em}h3{font-size:1.25em;margin-top:0.6em}img{max-width:100%;border-radius:8px;margin:1em 0}ul,ol{padding-left:1.5em}blockquote{border-left:3px solid #d1d5db;padding-left:1em;color:#6b7280}code{background:#f3f4f6;padding:2px 6px;border-radius:4px;font-size:0.9em}pre{background:#1e1e1e;color:#d4d4d4;padding:16px;border-radius:8px;overflow-x:auto}details{margin:0.5em 0;border:1px solid #e5e7eb;border-radius:6px;padding:0.5em 1em}summary{cursor:pointer;font-weight:600}</style></head><body><h1>${note.emoji} ${note.title}</h1>${note.content}</body></html>`;
    res.setHeader("Content-Type", "text/html");
    res.setHeader("Content-Disposition", `attachment; filename="${note.title.replace(/[^a-zA-Z0-9 ]/g, "")}.html"`);
    res.send(html);
  } else {
    const turndownService = new TurndownService({ headingStyle: "atx", codeBlockStyle: "fenced" });
    const markdown = `# ${note.emoji} ${note.title}\n\n${turndownService.turndown(note.content || "")}`;
    res.setHeader("Content-Type", "text/markdown");
    res.setHeader("Content-Disposition", `attachment; filename="${note.title.replace(/[^a-zA-Z0-9 ]/g, "")}.md"`);
    res.send(markdown);
  }
});
```

```ts
app.post("/api/lesson-notes/ai-enhance", isAuthenticated, async (req, res) => {
  const schema = z.object({
    text: z.string().min(1).max(5000),
    action: z.enum(["improve", "fix-grammar", "make-shorter", "make-longer", "summarize", "explain"]),
  });
  const { text, action } = schema.parse(req.body);
  const prompts: Record<string, string> = {
    "improve": "Improve the writing quality of the following text. Keep the same meaning but make it clearer and more professional. Return only the improved text.",
    "fix-grammar": "Fix all grammar, spelling, and punctuation errors in the following text. Return only the corrected text.",
    "make-shorter": "Make the following text more concise while keeping the key points. Return only the shortened text.",
    "make-longer": "Expand the following text with more detail and examples while keeping the same topic. Return only the expanded text.",
    "summarize": "Write a brief summary of the following text in 2-3 sentences. Return only the summary.",
    "explain": "Explain the following text in simpler terms that a beginner would understand. Return only the explanation.",
  };
  const result = await aiService.generateText({
    model: "gpt-4o-mini",
    system: prompts[action],
    prompt: text,
```

**Auto-save**: Use a `useCallback` with a 1000ms debounce (using `setTimeout`/`clearTimeout`). Every time the editor content changes, wait 1000ms of inactivity, then call `PATCH /api/lesson-notes/:id` with the new content. Show a small "Saving..." indicator (text-xs, text-muted-foreground) in the top-right corner of the editor area that appears during save and changes to "Saved" with a `Check` icon for 2 seconds after successful save. `data-testid="lesson-notes-save-status"`.

**Slash command menu**: Implement a custom slash command menu that appears when the user types "/" at the beginning of a new line or after a space. This is a floating popup positioned below the cursor, with a list of these commands:

| Command label | Icon | Description shown | Action |
|---|---|---|---|
| Heading 1 | `Heading1` | "Large heading" | `editor.chain().focus().toggleHeading({ level: 1 }).run()` |
| Heading 2 | `Heading2` | "Medium heading" | `editor.chain().focus().toggleHeading({ level: 2 }).run()` |

The slash menu should:
- Appear as a floating card (`bg-popover border rounded-lg shadow-lg p-1 w-72`) positioned below the text cursor.
- Filter items as the user types after "/" (e.g., "/hea" shows only the three heading options).
- Navigate with arrow keys (up/down), select with Enter, dismiss with Escape.
- After selecting a command, delete the "/" and any filter text from the editor, then execute the command.
- `data-testid="lesson-notes-slash-menu"`.

Implementation approach for slash commands: Use a TipTap extension or listen for the `keydown` event on the editor. When "/" is typed, record the cursor position and open the menu. On each subsequent keystroke, update the filter. On Enter/click, run the command. On Escape or clicking outside, close the menu. Delete the "/..." text before running the command using `editor.chain().focus().deleteRange({ from: slashPosition, to: currentPosition }).run()`.

### 3.6 Mutations

All mutations use `apiRequest` from `@/lib/queryClient` and invalidate cache on success.

- **Create note**: `POST /api/lesson-notes`. Invalidate `["/api/lesson-notes"]`. Set returned note ID as `selectedNoteId`.
- **Update note**: `PATCH /api/lesson-notes/:id`. Invalidate `["/api/lesson-notes"]` and `["/api/lesson-notes", noteId]`.
- **Delete note**: `DELETE /api/lesson-notes/:id`. Invalidate `["/api/lesson-notes"]`. Set `selectedNoteId` to `null`.
- **Upload image**: `POST /api/lesson-notes/upload-image`. No cache invalidation needed (returns a URL).
- **AI enhance**: `POST /api/lesson-notes/ai-enhance`. No cache invalidation needed (returns text).
- **Restore version**: `POST /api/lesson-notes/:id/versions/:versionId/restore`. Invalidate `["/api/lesson-notes"]`, `["/api/lesson-notes", noteId]`, and `["/api/lesson-notes", noteId, "versions"]`.

### 7.3 Auto-save Debounce Implementation

```ts
const saveTimerRef = useRef<NodeJS.Timeout | null>(null);

const debouncedSave = useCallback((html: string) => {
  if (saveTimerRef.current) clearTimeout(saveTimerRef.current);
  setSaveStatus("saving");
  saveTimerRef.current = setTimeout(async () => {
    if (!selectedNoteId) return;
    try {
      await apiRequest("PATCH", `/api/lesson-notes/${selectedNoteId}`, { content: html });
      queryClient.invalidateQueries({ queryKey: ["/api/lesson-notes"] });
      setSaveStatus("saved");
      setTimeout(() => setSaveStatus("idle"), 2000);
    } catch {
      toast({ title: "Failed to save note. Your changes may be lost.", variant: "destructive" });
      setSaveStatus("idle");
    }
  }, 1000);
}, [selectedNoteId]);
```

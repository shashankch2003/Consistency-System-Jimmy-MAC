14-17. DATABASE TABLES
Create these tables (all with serial() PKs, text user FKs):
• channels: id, workspaceId (int), name, type ('project'/'direct'/'team'), projectId (int, nullable), createdAt
• messages: id, channelId (int), senderId (text — varchar user ID), content, createdAt
• task_comments: id, taskId (int, references teamTasks), authorId (text — varchar user ID), content,
parentCommentId (int, nullable), createdAt
• documents: id, workspaceId (int), title, content, createdBy (text), isWiki (boolean, default false), parentDocumentId
(int, nullable), sortOrder (int, default 0), createdAt, updatedAt

=== PART A: REMAINING STANDARD BLOCKS ===
1. EQUATION BLOCK: Renders LaTeX/KaTeX math using the katex package. Click to edit
raw LaTeX, blur renders formula. Inline equations with $formula$ syntax.
2. TABLE OF CONTENTS: Auto-generates from all headings on the page. Updates live as
headings change. Click entry scrolls to that heading. Indented by heading level.
3. MENTION (inline): Type @ to trigger user mention popup. Type [[ to trigger page
mention popup. Type @date to insert date picker. Mentions render as colored pill
links.
4. SYNCED BLOCK: Create from block menu "Sync this block". Produces a synced copy
that stays identical. Edit one, all copies update. Orange border indicator. "Unsync"
option.
5. TEMPLATE BUTTON BLOCK: Click creates pre-defined block structure. Configure:
button label + template blocks array. Example: "Add Meeting Notes" button creates
heading + date + attendees list + action items.
6. LINK TO PAGE: Type /link or paste internal page URL. Renders as a card with page
icon + title + preview text. Click navigates to page.
7. LINK PREVIEW: Paste external URL on its own line. Auto-fetches and renders rich
preview card (title, description, image, favicon).
=== PART B: 10 UNIQUE BLOCKS (not in competitors) ===
8. VOICE NOTE BLOCK: Type /voice. Shows record button (red circle). Click starts
recording (animated pulse, timer counting). Click stop saves audio. Shows waveform
visualization + play/pause + duration. "Transcribe" button generates text
transcription below (mock AI for now). Transcription is searchable. NOTE: Voice
recording requires HTTPS and user microphone permission — this will work after
deployment but may not work in Replit's dev preview.
9. SKETCH BLOCK: Type /sketch. Opens canvas (h-[300px] bg-white border rounded-lg).
Tools: pen (3 sizes), eraser, color picker (8 colors), undo, redo, clear. Draw with
mouse/touch. Save as SVG in block state.
10. FLASHCARD BLOCK: Type /flashcard. Card with front/back. Click "Flip" toggles (3D
flip animation 300ms). Edit front (question) and back (answer). "Add to study deck"
link. Multiple flashcards stack as a deck.
11. PROGRESS TRACKER BLOCK: Type /progress. Shows: label input "Sprint Progress" +
current value slider (0-100) + color selector (green/blue/orange/red). Renders as
rounded bar with percentage text + label.
12. READING LIST BLOCK: Type /readinglist. Add URL input. Shows card: favicon +
title + domain + "Read"/"Unread" toggle badge. Stack multiple URLs as a list.
13. TIMELINE BLOCK: Type /timeline. Horizontal scrollable timeline. Each entry: date
+ title + description. "Add entry" button. Entries shown as dots on a horizontal
line with content below.
14. POLL BLOCK: Type /poll. Question input + option inputs (2-6 options). "Add
option" link. Shows results bar chart after voting. Click option to vote. Show
counts.
15. CODE RUNNER BLOCK: Type /run. Code editor (like code block) + "Run" button
below. Executing shows output in a result area below code. Support JavaScript
execution using a sandboxed iframe with srcdoc (do NOT use eval() as it may be
blocked by CSP). Display console.log output captured from the iframe.
16. BLOCK REACTIONS: Any block. Hover shows small "+" emoji button (bottom-right of
block). Click opens emoji picker (12 common emojis). Selecting adds reaction.
Reactions display below block as pills with count. Click existing reaction to
add/remove your vote.
17. BLOCK BOOKMARKS: From block menu select "Bookmark this block". Block saved to
user's bookmark collection (viewable from sidebar). Bookmarks reference the original
block and page.
